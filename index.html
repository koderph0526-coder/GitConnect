How to connect Git Hub to VSC

Step 1

Open Terminal in VSC after making certain that Git is downloaded and installed on the computer in question. Then switch from Powershell to Git Bash using the + with the arrow down.

Step 2 - the "tricky" part

Enter the code in the terminal "$ git config --global user.name exNameThatYouUseInYourGitAccount" and press enter, on the next line enter "$ git config --global user.email exTheEmailUsedInGitHub" and press engter again

Step 3

Then open the "Source" Tab in VSC and press "Initialize Repository", followed by pressing the + plus sign on each file from the project you want to upload or update from VSC, write "InitialCommit" as comment for the first entry, press Commit.

NB! If prompted in the VSC searchbar choose which GitAccount to send the repository to, I personally default to Private upload as the repository can be made public in GitHUb at a later occasion.
